# PolyCat & Poly‑Ontological Hubs — Complete Formalization (SMM‑Aligned)

## Abstract

We formalize **PolyCat**, a fibred category of poly‑ontological objects parameterized by a gauge of Ontological Primes, together with the theory of **hubs** (coherent law‑initial objects) and their algebra. We construct the gauge fibration \(\pi:\mathbf{PolyCat}\to\mathbf G\) with **extension–restriction** adjunctions \(E_p\dashv R_p\), prove **hub initiality** on finite objects, build the **projector algebra** \(\mathcal A_P\) and identify the **maximal envelope** \(\Omega_P^{\max}\) as the **Karoubi + inductive‑limit closure** of the fundamental cell, prove the **Rank Theorem** (max visible rank = number of odd primes in the gauge), and restate **SMM terminality** (homogeneity) over Law. We end with operational hub signatures (including 12,288 and higher hubs) and Lean‑level interfaces for fibers, hubs, adjunctions, and projector‑algebra modules.

## 1. Gauge Lattice and the PolyCat Fibration

- **Objects:** finite sets of primes \(P\subseteq \{2,3,5,7,\dots\}.\)
- **Morphisms:** inclusions \(i:P\hookrightarrow Q\). We write \(Q=P\cup\{p\}\) for a one‑step extension by an odd prime \(p\notin P\).

Fix a gauge \(P\). Let \(\Theta_P\) be the commuting projector family (receipts):

- structural (page \(2^4\), byte \(2^8\)),

- holography \(\Phi\),

**Fiber category.**

- **Objects:** \(P\)-coherent systems (harmonic points) \(X\) with budgets 0 under \(\Theta_P\).
- **Morphisms:** maps \(f:X\to Y\) commuting with every projector in \(\Theta_P\) (proof‑carrying lawful transforms).
- **Monoidal:** symmetric monoidal \((\otimes,\mathbf 1)\) via UN‑convolution; unit \(\mathbf 1\) (HUB(0)).

Let \(\mathbf{PolyCat}:=\bigsqcup_{P}\mathbf{PolyCat}_P\) and define \(\pi(X\in\mathbf{PolyCat}_P)=P\). For a covering inclusion \(i:P\hookrightarrow Q=P\cup\{p\}\), define:

- **Restriction** \(R_p:\mathbf{PolyCat}_Q\to\mathbf{PolyCat}_P\): forget \(\Pi_p\) (drop the \(p\)‑phase law), identity on underlying maps.
- **Extension** \(E_p:\mathbf{PolyCat}_P\to\mathbf{PolyCat}_Q\): minimally extend the boundary (e.g., cycle tiling) and enforce \(\Pi_p\); on arrows, retain the same map checked against the enlarged projector family.

**Adjunction (extension–restriction).** For all \(X\in\mathbf{PolyCat}_P\), \(Y\in\mathbf{PolyCat}_Q\), there is a natural bijection \(\mathrm{Hom}_{\mathbf{PolyCat}_Q}(E_pX,\,Y)\;\cong\;\mathrm{Hom}_{\mathbf{PolyCat}_P}(X,\,R_pY),\) whence \(E_p\dashv R_p\). The counit \(\varepsilon_Y:E_pR_pY\to Y\) is cartesian over \(i\), making \(\pi\) a Grothendieck **fibration**.

## 2. Hubs and Initiality on Finite Objects

Let \(\mathcal A_P\) be the unital \*‑algebra generated by the boundary symmetry algebra (shifts on the \(2^{12}\) torus) and the idempotents in \(\Theta_P\), modulo projector identities and commutation relations. Write \(e_P\) for the central idempotent that enforces all receipts (the product of the idempotents).

### 2.2 Hub definition

The **hub** \(\Omega_P\) is the coherent object represented by the regular coherent module \(M_P:=\mathcal A_P e_P\). Concretely, its boundary cardinality is \(|\Omega_P|=2^{12}\cdot\prod_{p\in P\setminus\{2\}}p\qquad(48\times 256\text{ for }\{2,3\};\;240\times 256\text{ for }\{2,3,5\}).\)

### 2.3 Initiality (finite objects)

**Theorem (Hub initiality).** For any **finite** \(P\)‑coherent object \(X\) (equivalently, a finite \(\mathcal A_P\)‑module), there is a **unique** coherence‑preserving morphism \(\Omega_P\to X\).

*Proof.* In modules, \(\mathrm{Hom}_{\mathcal A_P}(\mathcal A_P e_P, X)\cong e_P X\). Since \(X\) is coherent, \(e_P\) acts as identity, so \(e_P X\cong X\). Transporting back yields the unique morphism in \(\mathbf{PolyCat}_P\). \(\square\)

## 3. Maximal Envelope: Karoubi + Inductive‑Limit Closure

### 3.1 Karoubi envelope

Let \(\mathrm{Kar}\langle\Omega_P\rangle\) be the idempotent (Karoubi) completion of the additive subcategory generated by \(\Omega_P\): objects are pairs \((\Omega_P, e)\) with \(e^2=e\in\mathrm{End}(\Omega_P)\).

### 3.2 Cycle unrolling

Let \(u_m:\Omega_P\to\Omega_P^{(m)}\) be the inclusion obtained by **unrolling** the fairness cycle by a factor \(m\) (replace \(C_{\prod p\cdot 256}\) by its \(m\)-fold cover). The \(u_m\) form a filtered system.

### 3.3 Maximal envelope

**Definition.** \(\displaystyle \Omega_P^{\max}:=\varinjlim_m \big(\Omega_P^{(m)}\big)\) **inside** \(\mathrm{Kar}\langle\Omega_P\rangle\). Equivalently, \(\Omega_P^{\max}\) is the regular module of the von Neumann closure \(\mathcal M_P:=\mathcal A_P''\) acting on the boundary Hilbert space.

**Theorem.** \(\boxed{\;\Omega_P^{\max}=\text{Karoubi completion of }\Omega_P\ \text{closed under filtered colimits of unrolling}\;}.\)

*Proof (sketch).* Any extent‑expansion at fixed law is built from projector‑stable retracts (Karoubi) and cycle covers (filtered colimits). Conversely, any such object still carries the same \(\mathcal A_P\) action and remains \(P\)‑coherent. The von Neumann characterization identifies these closures. \(\square\)

## 4. Rank Theorem

Let \(P^*:=P\cap\{3,5,7,\dots\}\) and define **visible rank** as the number of UN atoms in an atomic \(\otimes\)‑factorization inside \(\mathbf{PolyCat}_P\).

**Theorem (Rank bound and sharpness).** For any \(X\in\mathbf{PolyCat}_P\), \(\mathrm{rank}(X)\ \le\ |P^*|,\quad \text{and this bound is attained on }\Omega_P.\)

*Proof (sketch).* The commuting fairness projectors \(\{\Pi_p\}_{p\in P^*}\) admit joint spectral decomposition into at most \(|P^*|\) central blocks. Each UN atom must live in a distinct block; otherwise receipts conflict at \(\beta=0\). For sharpness, choose CRT‑separated congruence classes realizing \(|P^*|\) atoms in \(\Omega_P\). \(\square\)

**Theorem (Terminality).** Any law‑equalizing strong symmetric‑monoidal realization \(G:\mathbf{PolyCat}_P\to\mathcal M_P\) admits a **unique** monoidal natural isomorphism \(G\Rightarrow F_{\mathrm{std}}\).

## 7. Lean‑Level Interfaces (sorry‑free signatures)

Minimal, composable interfaces to encode: (i) fibers, (ii) hubs, (iii) \(E_p\dashv R_p\), (iv) projector‑algebra modules. (Proofs can be added as instances.)

```lean
/-- Finite prime sets as gauges; inclusions form the lattice G. -/
structure Gauge := (carriers : Finset ℕ) (all_prime : ∀ p ∈ carriers, Nat.Prime p)

/-- Fiber objects: P‑coherent systems with commuting projectors. -/
structure PolyObj (P : Gauge) :=
  (U    : Type)
  (actR : U → U)   -- resonance
  (actC : U → U)   -- fairness (product over odd p ∈ P)
  (actΦ : U → U)   -- holography
  (actRy: U → U)   -- Reynolds
  (laws_comm : actR ∘ actC = actC ∘ actR ∧ actR ∘ actΦ = actΦ ∘ actR ∧
               actC ∘ actΦ = actΦ ∘ actC ∧ actRy ∘ actR = actR ∘ actRy ∧
               actRy ∘ actC = actC ∘ actRy ∧ actRy ∘ actΦ = actΦ ∘ actRy)

structure PolyHom {P} (X Y : PolyObj P) :=
  (map  : X.U → Y.U)
  (presR : Y.actR ∘ map = map ∘ X.actR)
  (presC : Y.actC ∘ map = map ∘ X.actC)
  (presΦ : Y.actΦ ∘ map = map ∘ X.actΦ)
  (presRy: Y.actRy ∘ map = map ∘ X.actRy)

/-- Category structure (omitted laws can be supplied as instances). -/
structure PolyCat (P : Gauge) :=
  (Hom   : PolyObj P → PolyObj P → Type := PolyHom)
  (id    : ∀ X, Hom X X)
  (comp  : ∀ {X Y Z}, Hom X Y → Hom Y Z → Hom X Z)

/-- One‑step cover P ↪ Q = P ∪ {p}. -/
structure Cover (P Q : Gauge) := (incl : P.carriers ⊆ Q.carriers)

/-- Restriction (forget the new law) and extension (insert the new law). -/
structure Restrict (P Q : Gauge) :=
  (onObj : PolyObj Q → PolyObj P)
  (onHom : ∀ {X Y}, PolyHom X Y → PolyHom (onObj X) (onObj Y))

structure Extend (P Q : Gauge) :=
  (onObj : PolyObj P → PolyObj Q)
  (onHom : ∀ {X Y}, PolyHom X Y → PolyHom (onObj X) (onObj Y))

/-- Adjunction between extension and restriction. -/
structure Adjunction {P Q} (E : Extend P Q) (R : Restrict P Q) :=
  (homEquiv : ∀ X Y, (PolyHom (E.onObj X) Y) ≃ (PolyHom X (R.onObj Y)))

/-- Hub with initiality on finite objects (interface). -/
structure Hub (P : Gauge) :=
  (obj : PolyObj P)
  (initial_finite : ∀ (X : PolyObj P), (Fintype X.U) → ∃! (u : PolyHom obj X), True)

/-- Projector algebra and coherent module interface. -/
structure ProjAlg (P : Gauge) := (A : Type) (e : A)
structure AModule (A : Type) := (V : Type) (act : A → V → V)
structure CohModule (P : Gauge) := (alg : ProjAlg P) (M : AModule alg.A)

/-- Maximal envelope witness (Karoubi + inductive‑limit closure). -/
structure MaxEnvelope (P : Gauge) :=
  (base   : Hub P)
  (Ωmax   : PolyObj P)
```
